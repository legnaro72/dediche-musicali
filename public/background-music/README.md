# Musica di sottofondo PWA

Inserisci qui il file MP3 leggero usato come musica di sottofondo.

Percorso predefinito:

```json
{
  "backgroundMusic": "/dediche-musicali/background-music/main.mp3",
  "audioDefaultEnabled": true,
  "audioVolume": 0.3
}
```

Per cambiare brano, volume o comportamento iniziale modifica `public/pwa-config.json`.
Consigliato: MP3 breve e leggero, ottimizzato per mobile, in loop continuo.
